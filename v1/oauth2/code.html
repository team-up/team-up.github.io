OAUTH2

Authorization Code Grant

1. code 요청

GET /oauth2/authorize
파라미터타입필수설명
response_typeStringYcode로 고정
client_idStringY어플리케이션 등록 후 발급받은 id
redirect_uriStringYcall back url
어플리케이션 등록할때 등록한 url
scopeStringN권한 관리용 scope , 로 구분합니다. ex) read,post
stateStringNcsrf 방지용
요청예
https://auth.tmup.com/oauth2/authorize?response_type=code&client_id=:id&redirect_uri=:url
결과
사용자가 로그인 하지 않았으면 로그인 페이지로 이동 합니다.
로그인 성공하면 동의 페이지로 이동합니다.
동의 후 설정한 redirect_uri 로 이동 합니다.
파라미터타입필수설명
codeStringY로그인 성공하면 반환되는 code
이후 token 발급 과정을 진행 하면 됩니다
stateStringNcsrf 방지용

2. token 요청

GET, POST /oauth2/token
파라미터타입필수설명
grant_typeStringYauthorization_code 고정
client_idStringY어플리케이션 등록 후 발급받은 id
client_secretStringY어플리케이션 등록 후 발급받은 secret
codeStringY발급받은 code
redirect_uriStringYcall back url
어플리케이션 등록할때 등록한 url
요청예
https://auth.tmup.com/oauth2/token? grant_type=authorization_code&code=:code&client_id=:id&redirect_uri=:url&client_secret=:secret
