1. Api 사용하기

발급 받은 token은 http header 에 넣어주거나 api 호출할때 parameter 로 넣어줍니다.
https://auth.tmup.com/v1?token=eeea79faac90473bdbaf9c807b3a1816fa115b20738f92ea6aaa93acce4d76aa
HTTP Header 에 아래와 같이 넣어줍니다. 
GET /v1 HTTP/1.1
Host: auth.tmup.com
Authorization : bearer eeea79faac90473bdbaf9c807b3a1816fa115b20738f92ea6aaa93acce4d76aa
