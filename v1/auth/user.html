회원

회원 api

1. 회원 정보 조회

GET /v1
token 으로 회원 index 조회
GET /v1/user
로그인된 사용자 정보 조회
POST /v1/user
사용자 정보 업데이트 
{
  "birthday": "1999-01-01",
  "is_lunar": false,
  "message": "하하하하",
  "mobile": "010-1111-1111"
}
GET /v1/user/:index
내가 속한 팀의 다른 회원 정보
GET /v1/user/teams
내가 속한 팀정보
GET /v1/user/team/:team index
내가 속한 팀별정보
POST /v1/user/team/:team index
팀별 내 정보 수정 
{
  "message": "하하하하",
  "mobile": "010-1111-1111",
  "phone": "02-2222-2222"
}
